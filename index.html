我們蒐集的資料將用於：

  • 提供與維護本App服務

  • 改善產品與使用體驗
  • 傳送通知、優惠或活動資訊（您可隨時取消）

4. Firebase Cloud Messaging（FCM）推播通知使用說明

本App使用 Google Firebase 所提供之 Firebase Cloud Messaging（FCM） 用於推播通知功能。

FCM 蒐集的資料

  • 裝置推播 Token（Device Token）

這些資料如何使用？

  • 僅用於向您的裝置發送應用內通知、課程提醒或重要公告。
  • 此資料無法用於個人身分識別。
  • 不會與第三方共用，也不會用於廣告投放。

為何需要 FCM？

  • 確保您能接收課務相關通知。
  • 便於學校行政或系統向使用者傳遞必要資訊。

FCM 資料安全

  • 資料透過加密方式傳輸。
  • FCM 所收集之裝置識別資訊僅於 Firebase 系統用途，不進行個資推斷。
  • Google Firebase 遵循 GDPR 與 ISO/IEC 27001 等國際安全標準。
